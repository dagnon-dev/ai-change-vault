# AI Change Vault Documentation

AI Change Vault is a local CLI for saving, indexing, searching, and reverting AI-assisted code
changes without requiring Git.
It runs in a Python environment on your machine and can manage any repository type, including
frontend projects and Node.js apps.

Core documents:

- `USAGE.md`: CLI examples.
- `ARCHITECTURE.md`: package structure and data flow.
- `AI_INTEGRATION.md`: protocol for coding agents.
