# AI Instructions for ai-change-vault

Read this before editing the project.

1. Look for AICV config in the project root.
2. Use `.aicv.config.yaml` as the project policy source.
3. Run a backup before making changes.
4. Make the requested change.
5. Validate the result.
6. Run a second backup and index the turn.
7. Use `aicv doctor` if the project is missing required files or directories.
